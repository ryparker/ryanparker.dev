## :rocket: Quick start

1. Clone this repository.

```shell
git clone https://github.com/ryparker/ryanparker.dev.git
```

2. Install dependencies

```shell
yarn install
```

3. Host the site locally

```shell
yarn develop
```

## Stack

| Technology | Use case                           |
| ---------- | ---------------------------------- |
| Gatsby     | Static site generator.             |
| React      | Modularized front end development. |
